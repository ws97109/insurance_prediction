🔗 模型架構

Deep Q-Network (DQN)

  • 輸入層: 85 個原始特徵
  • 隱藏層: 256 → 128 → 64 (ReLU + Dropout)
  • 特徵輸出: 32 個深度學習特徵

XGBoost Classifier

  • 輸入: 85 原始特徵 + 32 DQN 特徵 = 117 總特徵
  • Tree數量: 200
  • 最大深度: 6

📡 API 端點

1. 獲取模型資訊

GET /api/info

2. 單筆預測

POST /api/predict Content-Type: application/json Body: {"MOSTYPE": 33, "MAANTHUI": 1, ...}

3. 批量預測

POST /api/predict_batch Content-Type: application/json 或 multipart/form-data Body: [{"MOSTYPE": 33, ...}, {...}] 或 CSV 檔案

4. 特徵重要性

GET /api/feature_importance?top_n=20

5. 下載預測結果

POST /api/predict_batch/download 返回: CSV 檔案（包含 DQN 特徵資訊）
